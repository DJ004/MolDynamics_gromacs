### MolDynamics_gromacs
- `mdp_files`
  - Atomistic MD simulation (ATMD) parameters for EM, EQ, PROD - CHARMM36(m) ff
  - Coarse-gr MD simulation (CGMD) parameters for EM, EQ, PROD - MARTINI 2.2 ff
- `analysis` (analysis_scripts)
  - `do-order-multi_feb2023.py` - script to calc lipid order param for Martini 2 lipids
  - `count_num_of_lip-sol_*.sh` - script to quickly calc and print num of lipids and solvent molecules in a GRO/PDB file.
  - `pylipid_residence_time_analysis_r1-3` - directory containing a zip file with scripts and empty dirs ready for residence time analysis.
- `dssp`
  - contains a dssp executable and a deb file.
- `plotting` (more generic scripts will be added eventually)
- `sample_simulations` (in XTC format; along with corresponding GRO/PDB coordinates)
  - `MD_250ns_Lck_Tyr_Kinase`:
     - data generated by: Prakaash et al. Scientific Reports (2022). https://doi.org/10.1038/s41598-022-25603-6
     - data here is taken & post-processed from: https://archive.researchdata.leeds.ac.uk/968/ > ATMD_Lck-FL-closed.zip
